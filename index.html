Webscrapping

In this project, I developed a web scraping tool using Python to extract product information from Amazon, focusing on a specific item: the Russell Athletic Men's Dri-Power Short Sleeve Tee. By leveraging BeautifulSoup for parsing HTML and requests for fetching web pages, I automated the process of retrieving the product title and price.
